== Description ==
Download Featured Images will allow users to download featured images.

Major features in Download Featured Images include:

* Provide download button on featured images.
* A visitor can download featured images by a single click.
* Visitor will not be able to see download image without having featured images with posts/pages.

== Installation ==
1. Upload downloadfeaturedImages to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to Featured Settings and change the settings .
4. Go to Add New Post/Page/Custom Post type .
5. Set featured image .
6. Check Show Download Icon.


== Frequently Asked Questions ==

= How does it work ? =

1. Go to Featured Settings and change the settings .
2. Go to Add New Post/Page/Custom Post type .
3. Set featured image .
4. Check Show Download Icon.

= Will it appear without a featured images ? =

No , Download button will not be appear without featured images.

= How to exclude a post ? =
Just go to the Featured Settings and un check the post type.
